# <span style="color:rgb(0, 176, 240)">What is it</span>
Noun before the 「は」is the topic of the sentence.

# Some examples
* 私は研究者です
I as the topic. For me, 「研究者」 is me.

# <span style="color:rgb(0, 176, 240)">Some grammar:</span>
* Yes no question
A: ~<span style="color:rgb(192, 0, 0)"></span>は　<span style="color:rgb(192, 0, 0)">N</span>ですか。
B: はい、<span style="color:rgb(192, 0, 0)">N</span>です。
	いいえ、<span style="color:rgb(192, 0, 0)">N</span>じゃありません

<span style="color:rgb(192, 0, 0)">N</span> is noun. This is the confirmation question, to know that the infomation is true or wrong. When answer need to have
「はい」、「いいえ」

<span style="color:rgb(255, 255, 0)">Some example:</span>
ーーー
A:あの方はドイツ人ですか。
B:はい、ドイツ人です。
ーーー
A:あの方は医者ですか。
B:いいえ、医者じゃありません
* Questioning
A: ~は <span style="color:rgb(192, 0, 0)">Q</span>ですか。
B: <span style="color:rgb(192, 0, 0)">N</span>です。
<span style="color:rgb(192, 0, 0)">Q</span>: <span style="color:rgb(192, 0, 0)">Q</span> is an asking word such that (どこ、いつ、何~、...)
<span style="color:rgb(192, 0, 0)">N</span>: is a noun to answer <span style="color:rgb(192, 0, 0)">Q</span> 

<span style="color:rgb(255, 255, 0)">Some example:</span> 
A:Tueさんは何歳ですか。
		おいくつ
