# <span style="color:rgb(0, 176, 240)">What is it</span>
The 「に」keyword is <span style="color:rgb(255, 192, 0)">specify target of a verb</span>. It's like doing something "at"

# 「を」vs 「に」
**「を」 = The Object.** The verb "hits" this thing. **「に」 = The Target.** The verb "points" at this thing or "lands" on it.
