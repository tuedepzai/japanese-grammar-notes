# <span style="color:rgb(0, 176, 240)">Numbers smaller or less than 1</span> 
<span style="color:rgb(255, 255, 0)">Vocabulary</span>

- 1. 零 【れい】 – zero
- 2. ゼロ – zero
- 3. マル – circle; zero
- 4. 点 【てん】 – period; point
- 5. マイナス – minus

<span style="color:rgb(255, 255, 0)">How to read:</span>
Read each individual number and 「点」for the dot.
<span style="color:rgb(255, 255, 0)">For example</span> 
0.0021 = ゼロ，点、ゼロ、ゼロ、二、一。
-29 = マイナス二十九
